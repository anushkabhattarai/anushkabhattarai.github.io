## 🎯 Overview

This is my personal portfolio website — the public-facing introduction to who I am, what I do, and the projects I've shipped. It's a single-page experience designed to feel polished, fast, and true to the way I think about data work: clear, structured, and easy to navigate.

The site is **100% hand-coded** with HTML, CSS, and JavaScript. No React, no Tailwind, no build step — just clean web fundamentals you can open straight in a browser.

---

## ✨ Key Features

- 🎨 **Hero section** with animated mesh gradient, floating orbs, rotating skill chips, and a typewriter effect that cycles through what I turn data into
- 📱 **Fully responsive** layout — looks sharp on phones, tablets, and desktops
- 🧭 **Smooth-scroll navigation** with a sticky navbar and a hamburger menu on mobile
- 💼 **Experience timeline** with "current role" highlighting and duration badges
- 🛠️ **Tools & Skills grid** with custom SVG icons for Data Analysis, Visualization, Databases, Spreadsheets, ML, and Reporting
- 📊 **Stats callout** — quick-look numbers (projects shipped, certificates earned, academic standing)
- 🎓 **Education & Certificates** sections to showcase academic and professional credentials
- 📬 **Contact section** with direct links to email, LinkedIn, and GitHub
- 🎬 **Scroll-triggered animations** (`anim-up`, `anim-left`, `anim-right`, `anim-scale`) for a considered reveal as you read through
- 🖋️ **Google Fonts – Inter** (weights 300–900) for clean, modern typography
- ⚡ **Zero dependencies** — everything ships from three files

## ▶️ How to Run

### Option 1 — Open directly

Just double-click `index.html` — the site runs straight from the file system with no build step.

### Option 2 — Local web server (recommended for development)

```bash
cd Portfolio
python -m http.server 8000
```

Open [http://localhost:8000](http://localhost:8000) in your browser.

### Option 3 — Deploy it

This site is static, so it drops onto any static host with zero config:

- **GitHub Pages** — push to a repo, enable Pages in settings
- **Netlify** — drag-and-drop the folder, done
- **Vercel** — connect the repo, auto-deploys on every push

## 🎨 Design Philosophy

- **Clarity over cleverness.** Every section answers a specific question a visitor might have.
- **Motion with purpose.** Animations reveal content as you read, never for decoration's sake.
- **Mobile-first thinking.** The layout starts from a small screen and expands — not the other way around.
- **Performance by default.** No framework overhead, no external JS libraries, fast paint.
